# LED Blink Project

## Project Title
LED Blink using Arduino UNO

## Hardware Required
- Arduino UNO
- LED (5mm)
- 220Ω Resistor
- Breadboard
- Jumper Wires
- USB Cable

## Circuit Diagram Description
- Connect the LED anode (long leg) to Arduino Digital Pin 13.
- Connect the LED cathode (short leg) to a 220Ω resistor.
- Connect the other end of the resistor to the GND pin.
- Power the Arduino UNO using a USB cable.

## How to Upload Code
1. Open Arduino IDE.
2. Connect Arduino UNO to the computer.
3. Open the `led_blink.ino` file.
4. Select **Tools → Board → Arduino UNO**.
5. Select the correct COM Port.
6. Click the **Upload** button.
7. Wait for the message **"Done Uploading"**.

## Expected Output
- The LED connected to Pin 13 turns ON for 1 second.
- The LED turns OFF for 1 second.
- The blinking repeats continuously.

## Troubleshooting Tips
1. Check that the LED polarity is correct.
2. Ensure the 220Ω resistor is connected in series with the LED.
3. Verify that the correct Arduino board and COM port are selected.
4. Check all jumper wire connections.
5. Make sure the code uploads without errors.
